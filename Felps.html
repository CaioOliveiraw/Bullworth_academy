Disciplinas

 
 

Banco de Dados

Em nossa disciplina de Banco de Dados nossos alunos irão aprender a como criar e manusear um banco de dados.

A finalidade desta disciplina é fazerem os alunos terem o dominio em SQL Server e aprenderem a criar varios bancos.

Analise e Desenvolvimento de Sistemas

A disciplina de Analise e Desenvolvimento de Sistemas (ADS) é uma materia muito abrangente

A materia tem como disciplinas Logica de Porgamação, Matematica Aplicada e Linguagem de Progamação

Sistemas de Informação

O curso em Sistemas da Informação foi desenvolvido para que desde o inicio os alunos tenham a base para atuarem na area.

Os conteudos da disciplina vão de Hardware a Inteligencia Artificial, junto ao curso as materias de Engenharia de Software e Busines Intelligence
